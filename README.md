Descripción del proyecto:

	Este repositorio contiene el prototipo desarrollado para el Trabajo de Fin de Grado.
	La aplicación está implementada con Streamlit y permite visualizar y procesar datos relacionados con la capacidad de recursos.

Requisitos:

	Tener instalado Python 3.9 o superior.

Ejecución del prototipo:

	1. Instalar Streamlit (solo la primera vez):
		pip install streamlit
	2. Ejecutar la aplicación desde la carpeta del proyecto:
		streamlit run src/app.py
	3. El prototipo se abrirá automáticamente en el navegador. Si no lo hace, puede accederse manualmente a: http://localhost:8501

Contenido del repositorio:

	- Código fuente del prototipo.
	- Estructura del proyecto descrita en la memoria del TFG.
	- Versión final etiquetada como v1.0-final.
